# Video Editor App

A cross-platform Flutter application for video editing, built with a professional dark theme layout similar to popular mobile editing apps.

## Features

- **Video Playback**: Integrated video player with play/pause controls.
- **Timeline View**: Visual representation of the video timeline with a scrubber for navigating through the video.
- **Editing Tools**: A suite of editing tools including Split, Speed, Volume, and Delete operations.
- **Responsive Layout**: Designed to work gracefully on mobile, tablet, and desktop viewports, prioritizing the mobile editing experience.

## Tech Stack

- **Framework**: Flutter
- **Video Playback**: `video_player` package
- **Architecture**: Material 3 with a custom dark theme optimized for media editing.

## Getting Started

To run the application locally:

1. Ensure you have Flutter installed.
2. Clone the repository and navigate to the project directory.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to launch the app on your preferred device or emulator.
